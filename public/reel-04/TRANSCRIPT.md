# reel-04 — guion (voz del propio protagonista)

Mensaje reconstruido a partir de las tomas (transcripción local con
faster-whisper, es). Orden final problema → solución → cómo → CTA:

1. (c12) «Evitemos que esto pase. Así como con esta silla, pasa con miles de
   objetos que terminan en la basura.»
2. (c11) «Hemos creado Donnit para que puedas compartir los objetos que ya no
   usas con todos tus vecinos.»
3. (c05) «Con Donnit simplemente le sacas una foto y aparecerá disponible para
   todos los vecinos de tu barrio.»
4. (c14) «Descárgala y empecemos a limpiar juntos la ciudad.»

Se quitaron las pausas internas largas (c12 5.0–7.3s; c11 4.9–6.8s) sin cortar
palabras. Audio limpiado: highpass 120Hz + afftdn (denoise) + loudnorm I=-16.
