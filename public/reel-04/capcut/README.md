# Sube aquí tu export de CapCut

Arrastra el MP4 que hiciste en CapCut a esta carpeta `public/reel-04/capcut/`
(botón "Add file" → "Upload files" → commit a la rama
`claude/donnit-vertical-reels-a6vkqs`).

Cuando lo subas, avísame ("ya subí el de CapCut") y lo analizo:
- flujo/edición y qué modificar,
- si el audio/voz se entiende,
- y te sugiero música de fondo acorde para ponerle en CapCut.

Ojo: GitHub por web rechaza archivos > 25 MB. Si pesa más, exporta a 1080p /
calidad media, o dímelo y buscamos otra vía.
