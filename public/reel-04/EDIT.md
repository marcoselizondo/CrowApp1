# reel-04 — montaje "Rescata la silla" (Instagram Reel)

**Objetivo:** viralidad / descargas. 24.9s, 1080×1920 30fps, **sin música ni
subtítulos** (a petición). Edición directa con ffmpeg (esta rama no tiene el
proyecto Remotion).

**Historia:** camión lleno de muebles tirados → el chico rescata una silla del
camión → la usa → la pliega y se la lleva → CTA "Descarga Donnit / Dona lo que
no usas". Se prioriza la ACCIÓN (rescatar/cargar/plegar) para retención; los
planos hablando se usan cortos.
