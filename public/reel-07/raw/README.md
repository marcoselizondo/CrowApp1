# reel-07 — clips crudos

Sube **aquí** los vídeos/imágenes en crudo de este reel (`public/reel-07/raw/`).

- Cada reel vive en su propia carpeta → los clips no se mezclan con otros.
- `raw/` = crudos (los subes tú) · `seg/` = recortes/normalizados (los genero yo).
- Cuando termines de subir, avísame ("ya subí los de reel-07") y me vas dando las
  indicaciones (concepto, mensaje, orden, voz/música, formato) para montarlo.

Notas útiles:
- GitHub por web rechaza archivos > 25 MB; si algo pesa más, comprímelo o avísame.
- Puedo transcribir tu voz en local, editar con ffmpeg o montar en Remotion según
  convenga; tú marcas el estilo.
