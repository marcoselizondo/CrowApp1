# reel-08 — clips crudos

Sube **aquí** los vídeos/imágenes en crudo de este reel (`public/reel-08/raw/`).

- Cada reel en su carpeta → los clips no se mezclan con otros.
- `raw/` = crudos (los subes tú) · `seg/` = recortes/normalizados (los genero yo).
- Cuando termines de subir, avísame ("ya subí los de reel-08") y me das las
  indicaciones (concepto, mensaje, orden, voz/música, formato) para montarlo.

Puedo: transcribir tu voz en local, editar con ffmpeg o montar en Remotion
(subtítulos, kinetic, split-screen, música con ducking…), según convenga.
